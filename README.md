# 🎓 Student Performance Analysis

## 📌 Overview

This project analyzes student academic performance using data analysis techniques to identify patterns and factors affecting marks.

## 🛠️ Tools Used

* Python
* Pandas
* Matplotlib

## 📂 Dataset

* Student dataset with quiz marks, attendance, GPA, and exam scores

## 📊 Analysis Performed

* Data cleaning and preprocessing
* Feature engineering (Total Marks, Attendance %)
* Gender-wise performance analysis
* Attendance vs Marks analysis
* GPA vs Marks relationship
* Top student identification

## 📈 Visualizations

* Bar chart for Gender vs Marks
* Scatter plot for Attendance vs Marks
* Scatter plot for GPA vs Marks

## 🧠 Key Insights

* Higher attendance leads to better performance
* Previous GPA positively influences marks
* Top students perform consistently across exams
* Performance varies across different groups

## 🚀 How to Run

1. Install required libraries:
   pip install pandas matplotlib

2. Run the script:
   python student_analysis.py
